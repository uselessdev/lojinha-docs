---
title: "Autenticação"
description: "Autenticando sua aplicação"
---

Todos os endpoints da nossa API necessitam de autenticação, para fazer isso você precisa
criar [uma chave de api](https://lojinha.dev/settings).

Após sua chave ser criada você pode passar ela em um header de `Authorization`

```bash
curl https://lojinha.dev/api/... \
  -H "Authorization: pk_xxx"
```

Matenha sua chave segura e se necessário você pode revogar sua chave em **Configurações &rarr; Chaves de API**
