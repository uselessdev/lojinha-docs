---
title: "Criando carrinhos"
api: "POST https://lojinha.dev/api/v1/carts"
description: "Cria um carrinho"
authMethod: "bearer"
---

Carrinhos são úteis para que você tenha informações sobre carrinhos ativos e expirados
na sua loja, se você não precisa dessas informações, você pode criar um [pedido](/api-reference/endpoint/orders/create) ao invés de
um carrinho.

<Warning>
  Um produto só adicionado ao carrinho se quantidade disponível for menor
  ou igual a quantidade requisitada.

Caso nenhum produto da lista tenha disponibilidade a API retorna um erro com status 422.

</Warning>

### Body

<ParamField body="products" type="array">
  slug da coleção, retorna a primeira coleção encontrada.

  <Expandable name="products">
    <ParamField body="id" type="uuid" required>
      ID do produto.
    </ParamField>

    <ParamField body="quantity" type="number" required>
      Quantidade do produto para adicionar no carrinho.
    </ParamField>

  </Expandable>
</ParamField>

### Resposta

<ResponseField name="cart" type="object">
  <Expandable name="cart">
    <ResponseField name="id" type="uuid">
      ID do carrinho
    </ResponseField>

    <ResponseField name="store" type="string">
      ID da loja
    </ResponseField>

    <ResponseField name="customerId" type="string | null">
      ID do cliente
    </ResponseField>

    <ResponseField name="addressId" type="string | null">
      ID do endereço do cliente
    </ResponseField>

    <ResponseField name="price" type="number">
      Valor do carrinho
    </ResponseField>

    <ResponseField name="originalPrice" type="number">
      Valor original do carrinho
    </ResponseField>

    <ResponseField name="status" type="enum" default="PENDING">
      Status do carrinho. Neste endpoint o valor sempre será `PENDING`.
    </ResponseField>

    <ResponseField name="createdAt" type="string | null">
      Data de criação.
    </ResponseField>

    <ResponseField name="updatedAt" type="string | null">
      Data da última modificação
    </ResponseField>

    <ResponseField name="deletedAt" type="string | null">
      Data em que foi arquivado.
    </ResponseField>

    <ResponseField name="products" type="array">
      <Expandable>
        <ResponseField name="id" type="uuid">
          Id do produto
        </ResponseField>

        <ResponseField name="store" type="string">
          ID da loja
        </ResponseField>

        <ResponseField name="status" type="enum" default="DRAFT">
          Status do produto, pode ser: `DRAFT | ACTIVE`
        </ResponseField>

        <ResponseField name="name" type="string">
          Nome do produto
        </ResponseField>

        <ResponseField name="slug" type="string">
          Slug do produto
        </ResponseField>

        <ResponseField name="description" type="string | null">
          Descrição do produto em uma `string` com `HTML`
        </ResponseField>

        <ResponseField name="price" type="number | null">
          Preço atual do produto
        </ResponseField>

        <ResponseField name="originalPrice" type="number | null">
          Preço original do produto
        </ResponseField>

        <ResponseField name="quantity" type="number | null">
          Quantidade do produto em estoque
        </ResponseField>

        <ResponseField name="sku" type="string | null">
          Código do produto pra controle de estoque `Stock Keeping Unit`, pode
          ser o código de barras de um produto único
        </ResponseField>

        <ResponseField name="createdAt" type="string">
          Data de criação
        </ResponseField>

        <ResponseField name="updatedAt" type="string | null">
          Data da última modifição
        </ResponseField>

        <ResponseField name="deletedAt" type="string | null">
          Data em que que foi arquivado

          <Note>
            Pela API este dado sempre deve retornar `null`
          </Note>
        </ResponseField>

        <ResponseField name="order" type="object">
          Informações deste produto no carrinho.

          <Expandable>
            <ResponseField name="quantity" type="number">
              Quantidade deste produto neste carrinho.
            </ResponseField>

            <ResponseField name="price" type="number">
              Preço do produto no carrinho.
            </ResponseField>
          </Expandable>
        </ResponseField>
      </Expandable>

    </ResponseField>

  </Expandable>
</ResponseField>
