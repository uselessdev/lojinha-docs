## O que é o lojinha.dev?

[lojinha](https://lojinha.dev) é uma plataforma de e-commerce **headless** **open source**, ou seja,
nós fornecemos uma plataforma onde você cadastra seus produtos e consome seus dados
no front-end utilizando nossa API ou sdk independente do framework escolhido, você pode usar
[nextjs](https://nextjs.org), [vite](https://vitejs.dev/), [astro](https://astro.build/) ou
qualquer outro.
