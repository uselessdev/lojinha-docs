Antes de começar a cadastrar seus produtos, entendendo que os produtos serão
os que você vai vender na sua loja, tem algumas regras para que eles fiquem
disponíveis para você consumir os dados via API, os produtos só estarão dísponíveis
via API se duas situações forem atendidas:

1. O status do produto é **ativo**
2. A coleção em que o produto pertence não está **arquivda**
