## Chaves de API

Para acessar os da sua sua via API ou SDK você precisa ter uma chave de API, você pode criar sua chave
acessando https://lojinha.dev/settings.

<Warning>
  Sua chave só é mostrada para você logo após a criação, ela não deve ser
  compartilhada.
</Warning>
